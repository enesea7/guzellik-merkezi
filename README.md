# 💄 Güzellik Merkezi Web Sitesi

Bu proje PHP kullanılarak hazırlanmış basit bir güzellik merkezi web sitesidir.

## 🚀 Özellikler

- Anasayfa
- Hizmetler listesi
- Uzmanlar sayfası
- İletişim formu
- Randevu oluşturma
- Modern responsive tasarım

## 🛠 Kullanılan Teknolojiler

- PHP
- HTML
- CSS
- Bootstrap 5
- XAMPP

## 📂 Sayfalar

- index.php → Anasayfa
- hizmetler.php → Hizmetler
- uzmanlar.php → Uzmanlar
- iletisim.php → İletişim
- randevu.php → Randevu

2. xampp/htdocs klasörüne at
